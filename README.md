# Blog-Rearend
博客项目后端管理系统
# 项目介绍
核心框架: Spring Boot 2.2</br>
持久层框架: Mybatis Plus 3.0</br>
数据库: MySQL5.7 Redis</br>
数据库连接池: Alibaba Druid</br>
代码简化: Lombok</br>
文件上传: Jsch</br>
富文本插件: Simditor</br>
安全框架: Shiro</br>
日志管理: Slf4J</br>
模板引擎: Thymeleaf</br>
搜索引擎: ElasticSearch 7.2 (暂未开发)
